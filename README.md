# WebSockets server
Brief example of a WebSockets server.

## Usage
- Start the Docker container: `docker compose up -d --build`. The server will be available at `http://localhost:8080`
- Stop the Docker container: `docker compose down`

Find the client at https://github.com/arturomorarioja/js_ws_client.
