# WebSockets server
Brief example of a WebSockets server.

## Usage
1. Activate a Python virtual environment (Windows): 
```
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```
2. Run the server
    - `python ws_server.py` runs it on port 8080 by default
    - A parameter can be used to specify a different port: `python ws_server.py 5050`
3. Find the client at https://github.com/arturomorarioja/js_ws_client
